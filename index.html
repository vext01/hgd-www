The Opensource Multi-User Jukebox

What is HGD?

HGD (Hackathon Gunther Daemon) is a multi user jukebox suitible for hackathons, internet cafes, hackspaces etc.

The idea is that there is a central server connected to a set of speakers. Clients have user accounts on the server and may queue tracks from their own machines. This differs from the approach of MPD (for example), where there is a single centralised music library.

The concept was inspired by the LPD hack used at OpenBSD hackathons.

Features

  • Plays anything MPlayer does (including video).
  • Supports multiple user accounts.
  • Special "admin users" can pause, skip tracks etc.
  • Supports SSL encryption.
  • Written in portable C.
  • Released under a liberal ISC opensource license.
  • Simple and well documented network protocol
